       - `GetSubscriptions(database *mongo.Database) http.HandlerFunc`:
         - Extract user ID from JWT claims via userIDFromContext
         - Find all in CollSubscriptions where user_id matches, sorted by created_at descending
         - For each subscription, look up the channel from CollChannels by channel_id
         - Return JSON array of channels (not raw subscription documents) â€” this makes the frontend simpler
         - Return empty array `[]` if no subscriptions (not null)

       - `Subscribe(database *mongo.Database) http.HandlerFunc`:
         - Extract user ID from JWT claims
         - Get `{channel_id}` from chi URL param, parse to bson.ObjectID (return 400 if invalid)
         - Validate channel exists: FindOne in CollChannels by _id (return 404 if not found)
         - Create Subscription document: `{UserID, ChannelID, CreatedAt: time.Now()}`
         - InsertOne into CollSubscriptions
         - If `mongo.IsDuplicateKeyError(err)`: return 409 (already subscribed â€” client treats as success)
         - On success: return 201 Created

       - `Unsubscribe(database *mongo.Database) http.HandlerFunc`:
         - Extract user ID from JWT claims
         - Get `{channel_id}` from chi URL param, parse to bson.ObjectID
         - DeleteOne from CollSubscriptions where user_id AND channel_id match
         - Return 200 (idempotent â€” don't error if already unsubscribed)

       - `GetBookmarks(database *mongo.Database) http.HandlerFunc`:
         - Extract user ID from JWT claims
         - Find all in CollBookmarks where user_id matches, sorted by created_at descending
         - For each bookmark, look up the episode from CollEpisodes by episode_id (only if status="ready")
         - Return JSON array of episodes (not raw bookmark documents)
         - Return empty array if no bookmarks

       - `Bookmark(database *mongo.Database) http.HandlerFunc`:
         - Extract user ID from JWT claims
         - Get `{episode_id}` from chi URL param, parse to bson.ObjectID
         - Validate episode exists: FindOne in CollEpisodes by _id (return 404 if not found)
         - Create Bookmark document: `{UserID, EpisodeID, CreatedAt: time.Now()}`
         - InsertOne into CollBookmarks
         - If `mongo.IsDuplicateKeyError(err)`: return 409
         - On success: return 201 Created

       - `Unbookmark(database *mongo.Database) http.HandlerFunc`:
         - Extract user ID from JWT claims
         - Get `{episode_id}` from chi URL param, parse to bson.ObjectID
         - DeleteOne from CollBookmarks where user_id AND episode_id match
         - Return 200

       **Performance note:** GetSubscriptions and GetBookmarks do N+1 lookups (one per subscription/bookmark to fetch the channel/episode). For v1 with small user lists this is acceptable. If needed later, batch with $in operator.
