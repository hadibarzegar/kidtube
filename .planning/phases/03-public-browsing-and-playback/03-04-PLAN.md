<!-- From Plan 03-03 -->

VideoPlayer props:
```typescript
interface VideoPlayerProps {
  hlsSrc: string          // /hls/{episode_id}/master.m3u8
  subtitleSrc?: string    // subtitle_url from episode doc
  onEnded?: () => void    // triggers countdown overlay
}
```

Episode type:
```typescript
interface Episode {
  id: string
  channel_id: string
  title: string
  description: string
  order: number
  subtitle_url: string
  status: string
}
```

The watch page (from 03-03) fetches:
- Current episode: GET /episodes/{id}
- Channel: GET /channels/{episode.channel_id}
- Other episodes: GET /episodes?channel_id={episode.channel_id}

The "next episode" for autoplay is the episode with order = current.order + 1 in the same channel. If no next episode exists, no autoplay.
</interfaces>

       - This client component orchestrates the player + countdown + autoplay-next logic
       - Props: `{ episode: Episode; nextEpisode: Episode | null; channel: Channel }`
       - State: `showCountdown: boolean` (default false)
       - Import `VideoPlayer` from `@/components/VideoPlayerWrapper`

       - `onProceed` callback: `router.push('/watch/' + nextEpisode.id)`
       - `onCancel` callback: set `showCountdown = false`
       - Render:
         ```
         <div className="relative rounded-2xl overflow-hidden shadow-lg">
           <VideoPlayer
             hlsSrc={`/hls/${episode.id}/master.m3u8`}
             subtitleSrc={episode.subtitle_url || undefined}
             onEnded={onEnded}
           />
           {showCountdown && nextEpisode && (
             <CountdownOverlay
               nextEpisode={nextEpisode}
               onCancel={onCancel}
               onProceed={onProceed}
             />
           )}
         </div>
         ```
